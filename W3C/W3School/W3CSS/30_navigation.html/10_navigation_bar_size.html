Navigation Bar Sizes

Add the w3-small, w3-large, w3-xlarge or w3-xxlarge to change the size of the navigation (will change the font-size of each link).

Default:

Home Link 1 Link 2 Link 3

w3-large:

Home Link 1 Link 2 Link 3

w3-xlarge:

Home Link 1 Link 2 Link 3

w3-xxlarge:

Home Link 1 Link 2 Link 3

If you want more space between each link, instead of increasing the font-size, you could add a w3-padding-classes to each link:

Home Link 1 Link 2 Link 3
